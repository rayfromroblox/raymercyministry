# Ray Mercy Ministry Website

A modern static website for Ray Mercy Ministry charity organization.

## Features

- Next.js 14 with TypeScript
- Tailwind CSS styling
- Framer Motion animations
- Fully responsive design

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
